# wp-x3dom
x3Dom viewer for wordpress

use shortcode `[x3d id="post_id"]`

Attributes
`id` - id of the mode post type
`width` - changes the width of the x3d window
`height` - changes the height of the x3d window
`file` - used if you decide to upload a `.x3dom` file to post, default is set to false.  Must be set to true to utilize.  Only on HTTPS.
